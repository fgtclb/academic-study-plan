============
Installation
============

..  _installation-composer:

Install Academic StudyPlan with Composer
========================================

Install the extension via Composer:

..  code-block:: bash

    composer require 'fgtclb/academic-study-plan':'^2'

See also `Installing extensions, TYPO3 Getting started <https://docs.typo3.org/permalink/t3start:installing-extensions>`_.

..  _installation-classic:

Install Academic StudyPlan in Classic Mode
==========================================

Or download the extension from `https://www.fgtclb.com/ <https://www.fgtclb.com/>`_
and install it in the Extension Manager.
